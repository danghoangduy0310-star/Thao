    📐 Công thức Toán

    Đạo hàm
    Tích phân
    Lượng giác

    Quy tắc cơ bản

    (xⁿ)' = n·xⁿ⁻¹
    (c)' = 0
    (u+v)' = u' + v'
    (u·v)' = u'v + uv'
    (u/v)' = (u'v - uv') / v²

    Hàm thường gặp

    (eˣ)' = eˣ
    (ln x)' = 1/x
    (sin x)' = cos x
    (cos x)' = -sin x
    (tan x)' = 1/cos²x

    Hàm hợp

    [f(g(x))]' = f'(g(x))·g'(x)
    (√x)' = 1/(2√x)
    (aˣ)' = aˣ·ln a

    Ứng dụng

    PTT tại x₀: y-y₀=f'(x₀)(x-x₀)
    f'(x₀)=0 → cực trị
    f''(x)>0 → lõm lên

    Tích phân cơ bản

    ∫xⁿdx = xⁿ⁺¹/(n+1) + C
    ∫1/x dx = ln|x| + C
    ∫eˣdx = eˣ + C
    ∫aˣdx = aˣ/ln(a) + C

    Lượng giác

    ∫sin x dx = -cos x + C
    ∫cos x dx = sin x + C
    ∫1/cos²x dx = tan x + C
    ∫1/sin²x dx = -cot x + C

    Phương pháp từng phần

    ∫u dv = uv - ∫v du
    Chọn u: LOG-INV-ALG-TRI-EXP

    Tích phân xác định

    ∫ₐᵇf(x)dx = F(b) - F(a)
    S = ∫ₐᵇ|f(x)|dx (diện tích)
    ∫ₐᵇf = -∫ᵦₐf

    Hệ thức cơ bản

    sin²x + cos²x = 1
    tan x = sin x / cos x
    1 + tan²x = 1/cos²x
    1 + cot²x = 1/sin²x

    Cộng góc

    sin(a±b)=sin a·cos b±cos a·sin b
    cos(a±b)=cos a·cos b∓sin a·sin b
    tan(a±b)=(tan a±tan b)/(1∓tan a·tan b)

    Góc nhân đôi

    sin 2a = 2 sin a·cos a
    cos 2a = cos²a - sin²a
    cos 2a = 1 - 2sin²a
    cos 2a = 2cos²a - 1

    Tích → Tổng

    sin a·cos b = ½[sin(a+b)+sin(a-b)]
    cos a·cos b = ½[cos(a+b)+cos(a-b)]
    sin a·sin b = ½[cos(a-b)-cos(a+b)]

    Giới hạn nổi tiếng

    lim(x→0) sin x/x = 1
    lim(x→0) (1-cos x)/x² = 1/2
    lim(x→∞) (1+1/x)ˣ = e
    lim(x→0) (eˣ-1)/x = 1
    lim(x→0) ln(1+x)/x = 1

    Quy tắc L'Hôpital

    Dạng 0/0 hoặc ∞/∞:
    lim f/g = lim f'/g'
    (áp dụng khi giới hạn tồn tại)

    Vô cùng bé tương đương

    sin x ~ x (x→0)
    tan x ~ x (x→0)
    1-cos x ~ x²/2 (x→0)
    ln(1+x) ~ x (x→0)
    eˣ-1 ~ x (x→0)

    Liên tục & gián đoạn

    lim(x→a⁻)f = lim(x→a⁺)f = f(a)
    → f liên tục tại a
    Gián đoạn loại 1: giới hạn 1 phía tồn tại
